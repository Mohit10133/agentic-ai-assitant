# E-commerce Help

## Register account
To register, provide your name and email. Example: Register me with name John and email john@example.com.

## Registration tips
Use a valid email to receive order updates. Add a mobile number in profile settings for delivery alerts.

## Cancel order
You can cancel only if the order is not delivered. Provide order id. Example: Cancel order 10234.

## Return and refund policy
Return window is usually 7 days for eligible products in original condition. Refund is processed to original payment method after quality check.

## Apply coupon
At checkout, enter a valid coupon code such as SAVE20. Coupon must be active and within expiry date.

## Coupon rules
Only one coupon can be applied per order unless explicitly marked stackable. Some coupons require a minimum cart value.

## Order status
Ask with order id. Example: What is the status of my order 12345?

## Delivery timeline
Standard delivery takes 2 to 5 business days depending on city and warehouse availability.

## Change address
You can request delivery address update before shipment lock. Provide order id and new address.

## Payment options
Supported methods usually include UPI, cards, net banking, wallets, and cash on delivery where available.

## Escalation support
If your issue is unresolved, share order id and issue summary to create a support ticket.
